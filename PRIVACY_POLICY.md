# Privacy Policy for [Github-Repo-Radar]

**Last updated: June 1, 2026**

## 1. Data We Collect and How We Use It
This extension is designed to help users discover open-source projects on GitHub using AI. To provide this core functionality, we process the following:
* **Contextual Data:** We require the `activeTab` permission to read the URL of your current GitHub page solely to provide context for your search queries.
* **Search Queries:** The natural language text you input is securely transmitted to third-party Large Language Model (LLM) providers via API to generate search results. We do not store your search queries on our servers.

## 2. Analytics (PostHog)
We use PostHog to collect **strictly anonymous** usage data (such as how many users installed the extension and generic click events) to help us improve the user experience. 
* **No Personal Data:** We do not collect personally identifiable information (PII) such as your name, email, or GitHub account details.
* **Opt-Out:** You have full control. You can disable this anonymous data collection at any time in the extension's settings.

## 3. Data Storage
We use the browser's local `storage` API to temporarily cache your search results locally on your device. This data never leaves your browser and is used exclusively to improve loading speeds for your previous searches.

## 4. Data Sharing
We do not sell, rent, or trade any user data to third parties. Data is only shared with our designated LLM provider strictly for the purpose of processing your specific search requests.

## 5. Contact Us
If you have any questions about this Privacy Policy, please contact us at: [https://github.com/Loki-Tu/github-repo-radar/issues]
